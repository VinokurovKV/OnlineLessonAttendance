## Приложение для отслеживания присутствия на дистанционном уроке

Проект представляет собой клиент-серверное приложение, разработанное для контроля присутствия учащихся во время дистанционных занятий.

Цель проекта — предоставить учителю инструмент, позволяющий оперативно получать подтверждение присутствия учеников без использования сторонних систем видеосвязи и сложных интеграций.

### Основные возможности

- Клиент-серверная архитектура на Python
- Двусторонняя передача данных между учеником и учителем
- Автоматический и ручной вызов ученика
- Таймер автоматической проверки присутствия
- Интуитивно понятный графический интерфейс
- Поддержка одновременного подключения нескольких клиентов

### Технологии

- Python  
- Socket (сетевое взаимодействие)  
- Tkinter (графический интерфейс)  
- PyCharm (среда разработки)

Приложение ориентировано на простоту использования, минимализм интерфейса и высокую скорость отклика. Особое внимание уделено удобству для учителя и снижению количества лишних действий при работе с системой.
